# RichFaces-3.3.4.x
RichFaces 3.3.4に対してRF-7248の修正を適用したものです。  
そのままだとMavenビルドが通らないのでいくつかのpom.xmlを修正しています。  
公式ページにはRF-7248を適用したjarが公開されていません。  
自身でビルドとjar生成を行った際の備忘録のようなものです。
